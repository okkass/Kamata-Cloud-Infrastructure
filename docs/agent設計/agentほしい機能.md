- アップロードした OS イメージの削除。rm コマンドの発行

## ストレージプール

### 新規作成

- 新しいストレージを LVM 構築する
- iSCSI を構築する
- 返却値を別途検討(ストレージプールを識別するため)

### 更新

- iSCSI サーバーの設定を変更する窓口

## 仮想マシン

### 作成

- コマンドからじゃないと SG をアタッチできないらしい。
- 作成結果（成功/失敗）などを返してほしい
